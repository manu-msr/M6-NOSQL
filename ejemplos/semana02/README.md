# Semana 2: ejemplos de rendimiento y calidad de los datos

Esta semana contiene cuatro demostraciones guiadas:

- [Ejemplo 05](ejemplo05/README.md): comparación de una consulta por estado y
  fecha antes y después de crear un índice.
- [Ejemplo 06](ejemplo06/README.md): índice compuesto para filtrar por producto
  y estado y ordenar por fecha de inicio.
- [Ejemplo 07](ejemplo07/README.md): validación de los campos indispensables de
  un siniestro.
- [Ejemplo 08](ejemplo08/README.md): validación de arreglos de coberturas y de
  un documento anidado opcional.

Los ejemplos modelan el procedimiento y la interpretación. Las actividades que
resuelve el estudiantado se concentran en los retos 03 y 04.
