# Datos de la semana 1

Los ejemplos 01 a 04 utilizan la carga común ubicada en
[`../base/cargar_datos_base.js`](../base/cargar_datos_base.js).

Se conserva una sola fuente de datos para que las cuatro demostraciones trabajen
con los mismos identificadores, estructuras anidadas, arreglos y referencias.
El Reto 01 reutiliza esta carga para comprobar la configuración individual; el
Reto 02 dispone de un conjunto independiente dentro de su propia carpeta.
