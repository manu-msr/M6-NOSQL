# Retos de la semana 1

- [Reto 01](reto01/README.md): configurar la instancia individual, comprobar
  la base y completar una consulta breve sobre siniestros.
- [Reto 02](reto02/README.md): construir e interpretar un pipeline que resume
  frecuencia y monto por producto y cobertura afectada.

Los retos concentran los ejercicios de la semana y cuentan con una solución
docente separada. El Reto 01 reutiliza la base común para verificar el entorno;
el Reto 02 utiliza datos distintos de los ejemplos de agregación.
