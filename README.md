# Conceptos avanzados de bases de datos NoSQL

## 📘 Descripción

Este repositorio reúne los datos, ejemplos guiados y retos del módulo. Los
recursos están organizados por semana para acompañar el trabajo en clase y
facilitar que puedas repetir cada actividad dentro de AWS Academy Learner Lab.

A lo largo de seis semanas trabajarás con MongoDB para recuperar fundamentos de
bases de datos documentales y avanzar hacia la indexación, la validación de
esquemas, las consultas geoespaciales, las series de tiempo, la búsqueda textual
y la protección de datos. El recorrido culminará con la integración y defensa
de un proyecto de aplicación.
